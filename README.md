This project includes introduction of digital literacy and how to build a digital portfolio as a student on different types of websites . How to use coding platforms and how to make a google forms is also included in this project. This project tells us about the email etiquette and how to draw a professional emails using different types of examples, do's and don'ts checklists for social media, case studies about cybercrimes and their preventions.
Digital literacy is essential for every student in today's world. This project helps in building awareness about safe, responsible, and effective use of digital technologies, thereby tha risk of cyber threats and promoting a secure digital environment.
